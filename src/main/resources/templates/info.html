SeaSearch

В этот сервис можно авторизовываться только с помощью Telegram аккаунта. Он предназначен для того, чтобы анализировать свои личные сообщения. После авторизации Вы даете полный доступ к своим личным сообщениям, чтобы получать необходимые данные для корректной работы. Сообщения не будут нигде храниться, только во внутренней памяти веб-приложения, а после подведения итогов по анализу, они стираются из памяти. Здесь вы можете увидеть: 1. Облако слов из ваших самых популярных слов в употреблении. 2. Графика, которая иллюстрирует количество сообщений по дням. 3. Численные статистики по сообщениям. ВАЖНО! Telegram сервер позволяет получать только 100 сообщений в секунду, поэтому процесс может быть долгим. Вы можете запустить процесс, и заглянуть за результатом уже на следующий день.
